MediKita is a smart healthcare companion application developed independently to bridge the gap between patient medication adherence, emergency medical access, and peer support. Built using modern Android architecture, the app integrates local data persistence, real-time cloud computing, and physical hardware sensors to deliver a reliable, seamless medical ecosystem on a single mobile device. It is designed to empower users to take control of their health journeys while simplifying communication with localized healthcare providers and communities.

Milestone 01

Lab 1: UI & Layout Foundations

Designed the initial user interface for MediKita using Jetpack Compose layouts like Columns, Rows, and Boxes. I learned how to structure basic medical dashboards and patient profile views with composable components.

"Mastering modifiers for padding and alignment allowed me to create a clean, organized visual base for our health app."
Milestone 02

Lab 2: Dynamic State & Interaction

This lab focused on making MediKita dynamic by managing state with remember and mutableStateOf. I implemented interactive elements like medical symptom checkboxes, health metric input forms, and clickable appointment buttons.

"This taught me how Compose handles state recomposition to update UI elements instantly based on patient inputs."
Milestone 03

Lab 3: Material Design 3 Integration

I enhanced MediKita’s visual identity by applying Material Design 3 guidelines to establish a trustworthy healthcare color scheme, typography, and shape system.

"I integrated polished UI components such as TopAppBar, FloatingActionButton, and modern elevated cards to display medical alerts. This significantly elevated the application’s professionalism."
Milestone 04

Lab 4: Intent & Navigation Architectures

In Lab 4, I implemented robust navigation within MediKita using NavHost and NavController. I structured a clear navigation flow that seamlessly routes patients from the welcome screen to vital dashboards.

"This experience taught me how to securely pass medical data and arguments between composable screens while maintaining a healthy back-stack."
Milestone 05

Lab 5: Offline Persistence via Room

This lab introduced local data persistence to MediKita by integrating the SQLite-based Room Database. I built safe database entities, DAOs, and repositories to locally store patient records.

"Connecting this offline database with Compose UI using state flows ensures patient information remains accessible even without network connectivity."

Project Phase I — Core Application Layout

MediKita: MVP Foundation

Problem Statement: Patients and elderly individuals frequently struggle to manage their multiple medical appointments and daily medicine schedules due to fragmented tracking methods like paper notes. MediKita addresses this by providing a unified, localized mobile dashboard that organizes upcoming healthcare appointments and medication routines into a highly structured, intuitive user interface.

Lessons Learned: I learned how to scale a multi-screen application by establishing a robust navigation framework (NavHost) capable of passing persistent data securely between UI components. Developing this phase also deepened my understanding of state hoist routing, custom Material 3 styling components, and UI responsiveness across varying Android screen sizes.
Project Phase II — Connected Device Layer

MediKita: Cloud Sync & Hardware Sensors

Problem Statement: Effective personal health management requires real-time data monitoring and cross-device availability, which simple localized apps cannot provide. MediKita solves this by integrating web APIs for live medical resource lookups, device sensors and secure cloud databases to guarantee that vital patient information is synchronized safely and available anywhere, anytime.

Synthesized Summary

Learning Journey Epilogue

Over the course of developing MediKita from Lab 1 to Project 2, my skills evolved from designing basic layouts to architecting a complete, data-driven mobile ecosystem. Navigating through the implementation of reactive UI, complex navigation architectures, and local Room databases completely redefined my understanding of the Android lifecycle.

Integrating advanced capabilities like asynchronous web APIs, hardware sensor frameworks, and remote cloud infrastructure challenged me to think like an enterprise-level software engineer. Ultimately, this journey has not only made me proficient in Jetpack Compose and modern Kotlin practices, but it has also proven how digital technology can be leveraged to create impactful healthcare solutions.
